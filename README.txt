■ これは何？
  放射線治療の診療報酬（技術料・管理料・各種加算）の一連の総点数と
  窓口負担（1〜3割）の概算を、ブラウザ上で自動計算する学習用ツールです。
  ネット接続なしで動作する「オフライン版」です。

■ 使い方
  1. 同梱の「sim-rt-fee-calc-offline.html」をダブルクリックします。
     （お使いのブラウザ ― Chrome / Edge / Safari / Firefox 等 ― で開きます）
  2. 治療技術と加算を選ぶと、総点数と窓口負担の概算が自動で表示されます。
  3. インターネット接続は不要です。ファイル単体で動作します。

  ※ サーバーやJavaのインストールは一切不要です。
    計算ロジックはHTMLファイル内のJavaScriptで動作します
    （JavaScriptはブラウザ標準機能。プログラミング言語「Java」とは別物です）。

■ 点数の基準
  2026年6月改定（令和8年告示）ベースです。

■ ご注意（免責）

  ・点数は改定により変わります。最新情報は厚生労働省の告示等をご確認ください。
