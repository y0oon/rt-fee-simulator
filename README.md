# 放射線治療 算定点数シミュレータ（RT Fee Simulator）

放射線治療の一連の総点数と窓口負担の**概算**を、治療技術・加算を選ぶだけで自動計算する学習用ツールです。点数は **2026年6月改定（令和8年告示）** ベース。

> ⚠️ 本ツールは学習用の概算ツールです。実際の算定は最新の告示・通知・疑義解釈および審査支払機関の取扱いに従ってください。入院料・診察料・検査料等は含みません。

## デモ（オンライン版）

ブラウザでそのまま使えるトップページ（公開URLのホーム）:

- 公開URL: `https://y0oon.github.io/rt-fee-simulator/`

## ファイル構成

| ファイル | 用途 |
|---|---|
| `sim-rt-fee-calc.html` | シミュレータ本体（Web/iframe 埋め込み版） |
| `sim-rt-fee-calc-offline.html` | ダウンロード/オフライン配布版（日本語システムフォント fallback 付き・ネット未接続でも崩れない） |
| `index.html` | 公開URLのトップページ（本体と同一内容） |
| `build.sh` | 単一ソースから各版を生成・同期するビルドスクリプト |

## ビルド（版ズレ防止）

大本のソースを1つ編集し、以下を実行すれば各版が自動生成されます。

```bash
bash build.sh
```

- `sim-rt-fee-calc.html` … 本体（同期コピー）
- `sim-rt-fee-calc-offline.html` … タイトルに「（オフライン版）」付与＋日本語システムフォント fallback を追加

## 特徴

- 単一HTMLで完結（外部依存は Web フォントのみ。オフライン版はシステムフォントで代替）
- ガンマナイフ A400 短期滞在手術等基本料3 の包括算定に対応（A400 選択時は医療機器安全管理料2 など包括項目を自動で非表示）
- 小児加算（通則3）・IGRT・固定器具加算・呼吸性移動対策・粒子線/BNCT 加算などに対応
- 結果の CSV 保存・印刷/PDF 出力に対応

## 主な算定根拠（2026年6月改定）

- ガンマナイフ入院包括: A400 = 59,673点（生活療養時 59,569点。無床診療所は算定不可）
- 医療機器安全管理料2（B011-4）= 1,100点 … **A400 算定時は包括され別算定不可**
